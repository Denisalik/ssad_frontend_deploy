This repository is place for Doctorinna frontend project. It is built using next js framework on react, with redux state manager and material ui components. Current version is just layout of a SPA.

Later I will add:
 - new pages
 - components for questions
 - enhancement of UI
 - requests for API
 - implementation of questions components
 - result page after all questions are sent

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
